- 小鹤双拼

## 安装

- 备份原配置
- [最新鼠须管下载](https://dl.bintray.com/rime/squirrel/)
- 复制 2021 目录下文件到 ~/Library/Rime/
- 鼠须管菜单，点击部署与同步
